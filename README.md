# frases_motivacional

Projeto desenvolvido para aprendizado e melhorar habilidades no Flutter, esse projeto consiste em escolher através do widget Text que contém 6 tipos de frases motivacionais retirados do site o pensador o usuário ao clicar no botão FloatingActionButton é escolhido aleatoriamente uma das frases da lista, o limite para que as palavras caibam de maneira ideal no quadrado branco é de 217 caracteres. A aplicação poussi apenas uma tela com uma imagem de fundo e foi aplicado um Stack para colocar o texto por cima, foi colocado em prática o uso de fonte como referência do site Dafont.com.
